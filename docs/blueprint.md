# **App Name**: purbanchal hasta udyog

## Core Features:

- Product Listing & Search: Allow artisans to list their products with detailed descriptions and images, and allow customers to easily search and filter products.
- Secure Transaction: Secure transaction system that supports multiple payment options, like credit cards, UPI and netbanking.
- Artisan Profile: Allow artisans to create detailed profiles showcasing their skills, experience, and unique story.
- AI-Powered Craft Recommendation Tool: A recommendation tool that uses generative AI to suggest products or artisans to users based on their browsing history, preferences, and past purchases.
- Order Management: Allow artisans to efficiently manage orders, track shipments, and communicate with customers.
- Customer Reviews: Implement a customer review system where buyers can rate and provide feedback on products and artisans.

## Style Guidelines:

- Primary color: Terracotta (#E2725B) evoking the natural materials and earthy tones associated with handicrafts.
- Background color: Warm Beige (#F2EBD3) for a clean and inviting backdrop that enhances the handcrafted aesthetic.
- Accent color: Indigo (#4B0082) as a sophisticated, slightly contrasting hue that complements the terracotta.
- Headline font: 'Belleza', a humanist sans-serif. Body font: 'Alegreya', a humanist serif. This paring delivers personality alongside readability.
- Use hand-drawn style icons to represent categories, filters, and UI elements, to keep with the theme of handmade craftsmanship.
- Layout: Grid-based layout to showcase products in an organized manner, with featured artisans highlighted in a carousel.
- Subtle transitions and loading animations that evoke the gentle crafting process, like a loading spinner that resembles a potter's wheel.
